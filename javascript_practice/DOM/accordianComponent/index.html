What is JavaScript?
JavaScript is a programming language used to make web pages interactive.
What is DOM?
DOM stands for Document Object Model. It represents the structure of HTML as objects.
What is Event Bubbling?
Event bubbling is when an event propagates from child to parent elements.
What is Event Delegation?
Event delegation is handling events at a parent level instead of individual children.
What is React?
React is a JavaScript library for building user interfaces using components.
